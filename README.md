# Palmer-Penguins-Analysis
Final Exam for BANA7046: Data Mining 1 at University of Cincinnait.

- Conducted exploratory data analysis (EDA) on the Palmer Penguins dataset using data visualizations to identify patterns and relationships
- Built and evaluated classification models including Logistic Regression, CART and k-Nearest Neighbors (k-NN) to predict species
- Applied K-Fold Cross Validation to assess model performance and improve generalizability
